Instalación y ajustes iniciales
Juice-SSH Client

Lo primero que tenemos que hacer es irnos a la play store y buscar Juice-SSH Client, después de eso hacemos clic sobre la primera opción que nos aparece

Cuando la seleccionemos tenemos que darle a Instalar

Y esperamos a que termine la descarga

Cuando termine de descargar e instalar tenemos que darle a Abrir

Y ya estará listo para usar, pero antes tenemos que instalar ssh en el ordenador

Después de instalar Juice-SSH Client en el teléfono tenemos que irnos al ordenador y abrir el terminal

Y escribimos el comando que aparece en la pantalla para instalar ssh en Ubuntu

Escribimos la contraseña y empezara la instalación

Esperamos a que termine la instalación

Después de instalar ssh vamos a instalar nmap que sirve para ver si los puertos de ssh funcionan correctamente

Nos pedirá que confirmemos si queremos continuar, le decimos que sí y le damos a enter

Para usar nmap después de instalarlo tenemos que escribir nmap y la dirección 127.0.0.1 que es la ip que representa a la máquina que estamos usando

Si le damos a enter nos aparecerá todos los puertos que se están usando y que servicio lo está usando, si nos fijamos ssh está usando el puerto 22

Después de comprobar los puertos tenemos que saber la ip que tenemos para poder conectarnos

Después de preparar ssh en el ordenador tenemos que volver al teléfono y hacer clic sobre Manage Connections para añadir un usuario

En Conexiones aparece vacía porque nunca hemos usado esta aplicación, para añadir un usuario tenemos que darle al icono +

Le ponemos un nombre se usuario y le damos al tic para confirmar

En alias ponemos el nombre que vamos a usar, el tipo de dirección ssh y la identidad tenemos que elegir la que acabamos de crear ahora “Alejandro”

Y por último tenemos que añadir la dirección IP de nuestro servidor, cuando la añadamos le damos al tic para guardar

Cuando añadamos el usuario nos aparecerá en conexiones, para empezar la conexión tenemos que hacer clic sobre el

Antes de empezar la conexión con el servidor nos pedirá una verificación por ser la primera vez que nos conectamos.
Le damos a aceptar

Escribimos la contraseña de nuestro servidor y le damos a aceptar

Esperamos a que termine la conexión

Y ya estaríamos conectados con nuestro equipo Ubuntu mediando ssh



Crear carpeta Desde Android

Una vez dentro del servidor hacemos un ls para ver donde estamos ubicados y donde queremos crear la carpeta

Entramos en Desktop con cd para crear la carpeta

Dentro del escritorio usamos el comando Mkdir para crear la carpeta que nos pide en el ejercicio

Para comprobar que se ha creado correctamente la carpeta podemos usar ls

También podemos irnos al servidor y mirar en el escritorio a ver si se ha creado la carpeta
