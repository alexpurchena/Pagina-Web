SSH SERVER

Lo primero que tenemos que hacer es buscar ssh server en el buscador de la Play Store y hacer clic sobre la primera opción que nos aparece

Cuando hagamos clic sobre ella tenemos que darle a Instalar

Y esperamos a que termine la descarga

Cuando termine la instalación tenemos que darle a Abrir

Le damos a permitir y continuamos

Y ya tendríamos ssh server listo para usar en nuestro teléfono



Añadir usuario y entrar desde Windows

Dentro de ssh server, para añadir un usuario tenemos que irnos a la parte de debajo de la pantalla y pulsar sobre la pestaña Users

Dentro de la pestaña Users tenemos que pulsar sobre el icono que aparece una persona con un + abajo a la derecha de la pantalla

Le ponemos su nombre completo, el nombre de usuario y la contraseña después tenemos que pulsar sobre Add new para añadir a los lugares del teléfono donde los clientes van a poder acceder

Seleccionamos la tarjeta SD, por ejemplo, pero podemos seleccionar nuestro teléfono también

Seleccionamos la tarjeta SD en general y le damos a seleccionar

Después de añadir la tarjeta SD también añadí el almacenamiento de mi teléfono.
Cuando añadamos las carpetas a las que van a poder acceder tenemos que marcar la casilla Writable para que cuando accedan los usuarios a esas carpetas puedan crear y modificar las carpetas

Después de eso tenemos que activar la pestaña que está a la izquierda del nombre de usuario para habilitar al usuario

Cuando habilitemos al usuario podemos volver a la página principal y darle a start para iniciar el servicio ssh

Cuando le demos a iniciar nos saldrá la dirección Ip de nuestro teléfono y el puerto que está usando

Cuando terminemos de instalar ssh en el teléfono tenemos que irnos al ordenador y nos descargamos algo para conectar por ssh por ejemplo puTTy, nos vamos a su página oficial y hacemos clic “Aquí” para descargar el instalador

Y elegimos el instalador que vamos a usar de 32 o 64 bits según nuestro procesador, yo voy a elegir el de 64 bits

Una vez descargado hacemos doble clic sobre el para ejecutarlo

Cuando se inicie en la primera pantalla solo tenemos que darle a Next

Elegimos donde queremos instalarlo y le damos a Next

Aquí dejamos todo como esta y le damos a Install

Y por último solo tenemos que darle a Finish para terminar la instalación e iniciar puTTy

Cuando se inicie puTTy, en el apartado Host Name tenemos que poner la dirección IP que nos ha dado antes ssh server cuando lo hemos iniciado y justo al lado tenemos que poner el puerto que nos indicaba y le damos a Open para iniciar la conexión

Cuando se abra la terminar de puTTy lo primero que tenemos que hacer es poner el nombre de usuario del usuario que hemos creado anteriormente y después nos pedirá la contraseña del usuario

Después de poner el usuario y la contraseña ya estaríamos conectados con el teléfono por ssh

Si hacemos un ls en la terminal de puTTy podemos ver todas las carpetas de nuestro teléfono así podemos comprobar que nos hemos conectado correctamente



Crear carpeta en SSH Server

Para poder crear una carpeta desde puTTy tenemos que acceder a unas de las rutas que nos ha dado la aplicación ssh server cuando le hemos añadido la tarjeta sd y el almacemaniento interno de nuestro teléfono, porque si accedemos a otra ruta que no sea esa nos dirá que no tenemos permisos de escritura sobre ella

Hacemos un ls y comprobamos que hemos accedido al almacenamiento interno del teléfono

Cuando comprobemos que estamos en la ruta correcta podemos usar el comando mkdir para crear la carpeta que nos pide

Si volvemos hacer un ls después de ejecutar el comando mkdir comprobamos que nos aparece la nueva carpeta



Si nos vamos a la aplicación AirDroid o a los archivos de nuestro teléfono podemos comprobar que sea creado correctamente la carpeta
