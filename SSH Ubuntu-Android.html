SSH Ubuntu-Android

Lo primero que tenemos que hacer es abrir la aplicación ssh server en nuestro teléfono e irnos a la pestaña Users

En la pestaña Users le damos al icono del muñeco para añadir un nuevo usuario

Le ponemos nombre, nombre de usuario y contraseña y después pulsamos sobre Add new

Nos vamos a una carpeta del móvil y la seleccionamos para que el usuario que se conecte solo pueda tener permisos de escrituras sobre esa carpeta, después de seleccionar la carpeta le damos a seleccionar

Cuando creemos el usuario en el servidor tenemos que irnos al cliente y poner “ssh –p 222 alexubuntu@192.168.0.100” para conectarse con el servidor, cuando le demos enter tendremos que confirmarle si queremos continuar con la conexión le decimos que sí y por ultimo escribimos la contraseña

Y ya estaríamos conectados con nuestro teléfono

También podemos irnos al servidor y comprobar que estamos conectados

Una vez dentro del servidor tenemos que acceder a una de las rutas que hemos añadido cuando hemos creado en el usuario /storage/emulated/0/ y ya dentro de la carpeta donde tenemos permisos de escritura podemos crear la carpeta desdeubuntu con el comando mkdir

Si hacemos un ls podemos comprobar si se ha creado correctamente la carpeta
