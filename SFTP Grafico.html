SFTP gráfico

Lo primero que tenemos que hacer es irnos a la play store, buscamos la aplicación y le damos a instalar

Esperamos a que se instale

Una que se instale le damos a Abrir

Le damos a permitir para que pueda acceder a nuestros archivos y poder utilizar correctamente la aplicación

Después de aceptar los permisos la aplicación ya estaría lista para usar y conectarse al servidor.
Para empezar a usar la aplicación tenemos que darle al signo “Mas” para añadir un servidor y poder hacer la conexión

Añadimos la dirección IP del servidor al cual nos vamos a conectar y le damos a salvar

Podemos cambiarle el nombre para saber a qué servidor nos estamos conectando o podemos dejarlo así si sabemos que esa es su IP

Le damos a Ok y terminaremos con la configuración

Cuando lo tengamos todo configurado podemos mantener pulsado sobre la carpeta y nos dará unas opciones

Nosotros le vamos a dar a Editar para añadirle la ubicación de almacemaniento de nuestro teléfono

Le damos a External para elegir la tarjeta SD, si no disponemos de una podemos seleccionar el almacenamiento interno del teléfono

Seleccionamos la carpeta que queramos y le damos a seleccionar

Cuando tengamos seleccionada la carpeta le damos a Salvar

Una vez realizado todo lo anterior ya podemos pulsar sobre la carpeta para empezar la conexión con el servidor

Cuando pulsemos sobre ella nos pedira un nombre y una contraseña, tenemos que poner el usuario y la contraseña que tenemos en nuestro servidor

Después de escribir el usuario y la contraseña ya estaríamos conectados con el servidor, si le damos al icono del teléfono entraremos a la carpeta que anterior mente hemos añadido para que se descarguen en ella los archivos del servidor

He creado una carpeta para tener algo para copiar, para copiar un archivo tenemos que mantener pulsado el archivo hasta que nos salga el tic y después tenemos que darle a la flechita para arriba para subir el archivo al servidor

A continuación, nos pedirá que confirmemos que queremos subir el archivo, le damos a Ok

Esperamos a que termine y ya tendríamos la carpeta en la ruta que ponía en la pantalla anterior

Cuando copiemos la carpeta del teléfono al servidor tenemos que irnos al servidor, al escritorio por ejemplo y elegimos un archivo

Yo voy a seleccionar los usuarios que cree para moodle, después tenemos que darle a la flechita para abajo para que empiece la descarga

Confirmamos que queremos descargar dándole a Ok

Y los usuarios de Moodle nos aparecen en la carpeta que le hemos dicho antes y ya sabríamos todo lo necesario para usar esta aplicación
